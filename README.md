# Vanillalauncher

Tauri + React + TypeScript を組み合わせた Minecraft ランチャーの開発リポジトリ（Vite + Rust + Tauri）。

## 概要

- フロントエンド: React + TypeScript + Vite
- デスクトップ: Tauri (Rust)
- 開発フローに Bun を推奨
